# Workflow-Pflicht

- Vor jeder Code-Änderung: CLAUDE.md, docs/PRD.md und die
  relevante Datei in features/ lesen.
- Kein direkter Zugriff auf Code ohne vorherigen Spec-Check,
  außer bei explizit als "Hotfix" markierten Anfragen.
- Nach jedem /write-spec und /architecture: auf explizites
  "approved" warten, nicht eigenständig weitermachen.
- Status in features/INDEX.md und im jeweiligen Feature-Header
  immer synchron halten.
